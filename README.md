# 🚀 IoT Attack Detection using Machine Learning & Deep Learning

A complete cybersecurity project that detects and classifies IoT network attacks using multiple Machine Learning and Deep Learning models. This project focuses on improving threat detection accuracy in IoT environments using real-world datasets.

---

## 📌 Overview

With the rapid growth of IoT devices, security threats have increased significantly. This project applies various ML and DL algorithms to identify malicious network traffic and classify different types of attacks.

The system:
- Processes real IoT network traffic data  
- Performs data preprocessing and feature engineering  
- Trains multiple models  
- Evaluates performance using standard metrics  

---

## ⚙️ Technologies Used

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- TensorFlow / Keras  

---

## 📊 Dataset

- Dataset used: **RT_IOT2022**
- Contains IoT network traffic with labeled attack types  
- Includes both normal and malicious traffic  

---

## 🧠 Models Implemented

### Machine Learning Models
- Logistic Regression  
- Decision Tree  
- Random Forest  
- K-Nearest Neighbors (KNN)  

### Deep Learning Models
- Artificial Neural Network (ANN)  
- Multilayer Perceptron (MLP)  

---

## 🔄 Workflow

1. Data Extraction (ZIP handling)
2. Data Cleaning
   - Removed unnecessary columns  
   - Handled missing values  
3. Data Preprocessing
   - Label Encoding for categorical features  
   - Feature scaling using StandardScaler  
4. Train-Test Split  
5. Model Training  
6. Evaluation  

---

## 📈 Evaluation Metrics

Models are evaluated using:

- Accuracy  
- Precision  
- Recall  
- F1-Score  
- Confusion Matrix
