# RT_IOT2022 Attack Type Classification

A machine learning and deep learning project for network attack type
classification using the RT_IOT2022 dataset.

------------------------------------------------------------------------

## Overview

This project builds a complete pipeline:

-   load dataset from CSV
-   clean and preprocess data
-   encode categorical features
-   train ML and DL models
-   evaluate performance
-   visualize results
-   save trained models

------------------------------------------------------------------------

## Features

-   Clean and beginner-friendly workflow
-   Direct CSV input support
-   Multiple ML and DL models
-   Confusion matrix visualization
-   Performance comparison graphs
-   Model saving for deployment

------------------------------------------------------------------------

## Models Used

### Machine Learning

-   Logistic Regression
-   KNN
-   Random Forest
-   SVM

### Deep Learning

-   ANN
-   LSTM

------------------------------------------------------------------------

## Dataset

-   File: RT_IOT2022.csv
-   Target: Attack_type

## Workflow

1.  Load dataset
